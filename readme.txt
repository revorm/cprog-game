Welcome to cprog game!
 - A game written in C++ about writing a game in C++

The main goal of the game is to finish the third computer lab in the
C++ course at CSC.

To accomplish this you first need to come up with some ideas for the game.
Then you need the energy and patience to code it, and finally you need
to get ahold of the course examiner in order to get it graded and finish the game.

To start an automatic demonstration mode of the game pass the parameter "a"
to the game executable, "$ ./game a".

To play the game in interactive mode you type commands in the command prompt.
To find out more about which commands are available type "help".
